# UpSkills
Site live at: https://up-skills.herokuapp.com/
To try out:
Launching Backend:
```
git clone https://github.com/namandureja/upSkills.git
cd upSkills
npm start
```

Launching Frontend:
```
cd client
yarn start
```

Launching AI Backend:
```
pip install -r requirements.txt
python3 -m flask run
```
